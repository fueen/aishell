# aishell
**智慧售**

智能销售平台，轻松，简单，快速的为商家提供采购、进货、库管、销售、审批、权限、报表统计等信息操作

主要设计模块：基础模块、产品模块、员工模块、订单模块、报表模块等

## IDE

IDE(2019.2)

Jdk1.8

MySql5.7

Tomcat8.0

## 技术栈

### 后端

Spring+SpringMvc+SpringDataJPA+Shiro+Highcharts 

### 前端

EasyUI+Jquery+Ajax

### 主要涉及知识点

表之间关系结构

Shiro权限配置

SpringDataJPA使用

报表复杂查询

Highcharts生成图表
